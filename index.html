About

What I do

I work primarily as a Full-Stack Developer, dividing my time between backend logic, frontend interfaces, and system integrations.

Backend
I build microservices and backend logic using Java (1.8 - 25) and Spring Boot. I handle database migrations with Liquibase, use Docker for containerization, and work with both relational (PostgreSQL, MySQL, Oracle) and non-relational databases (Cassandra, MongoDB).
Java Spring Boot Microservices SQL NoSQL
